## Features

| Feature | Description |
|---------|-------------|
| **Multi-Platform** | eBay, Poshmark, Mercari, Depop, FB Marketplace |
| **Smart Titles** | SEO-optimized titles with emoji hooks |
| **Descriptions** | Conversion-focused copy with trust signals |
| **Keywords** | Platform-specific tag suggestions |
| **Pro Tips** | Best posting times & listing strategies |

---

## Quick Start

```bash
git clone https://github.com/DeadManOfficial/listingpro-ai.git
cd listingpro-ai
npm install
npm run dev
```

Open `http://localhost:5173`

---

## Usage

1. Select your selling platform
2. Describe your product (brand, condition, size, color)
3. Click **Optimize Listing**
4. Copy optimized title, description & keywords
